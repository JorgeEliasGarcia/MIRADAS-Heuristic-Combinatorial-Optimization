# MIRADAS-Heuristic-Combinatorial-Optimization
Heuristic solution to the combinatorial optimization problem of the Gran Telescopio Canarias (GTC) robotic instrument, MIRADAS. Modeled as a no-wait Consistent Traveling Salesman Problem (CTSP1) variant for target prioritization, implemented in C++ with constructive and local-search heuristics and benchmarked on 756 TSPLIB instances.
